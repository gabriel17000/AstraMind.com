# AstraMind.com

Protótipo visual de plataforma SaaS de inteligência artificial.

Stack:
- React
- Vite
- TailwindCSS
- React Router
- Framer Motion
- Recharts
- Lucide React

## Como usar

1. Instalar dependências:
   ```bash
   npm install
   ```
2. Rodar o projeto:
   ```bash
   npm run dev
   ```
